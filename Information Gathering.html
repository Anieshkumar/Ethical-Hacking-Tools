Whois

Whois is a protocol that allows you to look up information about domain names and IP addresses, including ownership, registration, and contact details. It is commonly used in information gathering and reconnaissance during ethical hacking.

To use the Whois tool, you can either use a web-based Whois lookup service, or use a command-line tool like the 'whois' command in Linux or Windows.


To perform a Whois lookup on a domain name or IP address using the command-line tool, open a terminal window and type the following command:

whois example.com

This will display the registration and contact details for the domain name 'example.com'. You can also use the '-h' option to specify the Whois server to use, if the default server is not suitable for your needs.



FOCA

Maltego CE

Maltego CE (Community Edition) is a popular open-source tool used for information gathering and reconnaissance during ethical hacking. It allows users to visualize and understand complex relationships between different entities, such as organizations, people, and technologies, and can aid in identifying potential vulnerabilities and attack vectors.

Here's an example of how to use Maltego CE:

  1. 1.Download and install Maltego CE on your computer.
  2. 2.Launch Maltego CE and create a new graph.
  3. 3.Add an entity to the graph, such as a domain or social media profile.
  4. 4.Select the entity and choose a transform from the Transform Hub to run on it. A transform is a pre-built script that queries a data source and retrieves information about the entity.
  5. 5.Wait for the transform to complete and view the results in the graph. Maltego CE will automatically add any new entities and relationships to the graph.
  6. 6.Repeat steps 4 and 5 with other entities in the graph to build a comprehensive view of the target.


SpiderFoot

SpiderFoot is an open-source intelligence (OSINT) automation tool used for information gathering and reconnaissance. It automates the process of gathering intelligence about a target, including domains, IP addresses, email addresses, social media accounts, and more. It is designed to be used by both security professionals and non-technical users.

Here's an example of how to use SpiderFoot:

  1. 1.Download and install SpiderFoot on your computer.
  2. 2.Launch SpiderFoot and create a new scan profile.
  3. 3.Configure the profile with your target information, including the domain name, IP address, or other relevant details.
  4. 4.Run the scan and wait for SpiderFoot to complete the reconnaissance process. This may take several minutes, depending on the size of your target and the depth of the scan.
  5. 5.Review the results in the SpiderFoot dashboard, which will show you a detailed report of the intelligence gathered.
  6. 6.Export the results to a CSV file or other format for further analysis or reporting.


The Harvester

The Harvester is an open-source intelligence (OSINT) tool used for gathering email addresses, subdomains, virtual hosts, open ports, banners, and employee names from different public sources like search engines and social media platforms. It is commonly used by penetration testers, security researchers, and cybercriminals for reconnaissance and information gathering.

Here's an example of how to use The Harvester:

  1. 1.Download and install The Harvester on your computer.
  2. 2.Launch The Harvester and select the data sources you want to use for your search, such as Google, Bing, LinkedIn, or Twitter.
  3. 3.Configure your search options, including the domain or keyword you want to search for, the number of results to retrieve, and any other specific settings you want to use.
  4. 4.Run the search and wait for The Harvester to complete the process. This may take several minutes, depending on the size of your search and the number of data sources you selected.
  5. 5.Review the results in the output file generated by The Harvester, which will contain the email addresses, subdomains, or other information gathered during the search.

It is designed to automate the process of gathering and analyzing information from various sources, such as search engines, social networks, and domain name systems (DNS).

Here's an example of how to use Recon-ng:

Google Dorks

Google dorks are advanced search operators that can be used to search Google for specific information that is not easily accessible through a regular search. These dorks allow users to perform more targeted searches to find sensitive information, such as usernames, passwords, email addresses, and other data that can be used in a cyber attack.

Here are some examples of Google dorks and their usage:

  1. 1.site:example.com intitle:"login" - This dork searches for login pages on the specified website.
  2. 2.filetype:pdf site:example.com - This dork searches for PDF files on the specified website.
  3. 3.intext:"username" site:example.com - This dork searches for pages on the specified website that contain the word "username".
  4. 4.inurl:"/admin.php" site:example.com - This dork searches for pages on the specified website that contain "/admin.php" in the URL.
  5. 5.site:example.com intitle:"index of" - This dork searches for open directories on the specified website.


Metagoofil

Metagoofil is a tool used for extracting metadata from public documents (such as PDF, DOC, XLS, PPT, etc.) available on a target domain or IP address. The tool is useful for information gathering during penetration testing, vulnerability assessment, or red team engagements.

Here are the steps to use Metagoofil:

  1. 1.Install Metagoofil on your system. It is available for Linux and Windows.
  2. 2.Open the terminal and navigate to the Metagoofil directory.
  3. 3.Run the command 
    "python metagoofil.py -d [domain name] -t [filetype] -o [output directory] -l [limit]"
    to start the metadata extraction process.
    Replace [domain name] with the target domain name or IP address, [filetype] with the file type you want to extract metadata from (e.g. pdf, doc, xls, etc.), [output directory] with the location where the extracted metadata will be saved, and [limit] with the maximum number of files to extract metadata from.
  4. 4.Wait for the tool to complete the metadata extraction process.
  5. 5.Analyze the extracted metadata to gather information about the target domain or IP address.
