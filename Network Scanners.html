Network Scanners

*****Caution: Remember to always use these tools ethically and with Permission! *****

Nmap

Nmap is a powerful network exploration tool that allows you to discover hosts and services on a computer network, as well as create a map of the network.

Here's an example of how to use Nmap:

  1. 1.Download and install Nmap on your computer.
  2. 2.Open a command prompt or terminal and enter the command: nmap [target]
  3. 3.Replace [target] with the IP address or hostname of the target network or host that you want to scan.
  4. 4.Optionally, you can specify additional scanning options, such as the types of scans to perform, the ports to scan, and the level of verbosity.
  5. 5.Press Enter to start the scan.
  6. 6.Once the scan is complete, Nmap will display the results in the command prompt or terminal.

Here's an example of a basic Nmap command:

nmap 192.168.1.1

This command will scan the host with IP address 192.168.1.1 for open ports and display the results in the command prompt or terminal.


Zenmap

Zenmap is a graphical user interface (GUI) for the Nmap network scanner, which is used to discover hosts and services on a computer network. It is a free and open-source tool that is available for Windows, Linux, and macOS.

Here's an example of how to use Zenmap:


  1. 1.Download and install Zenmap on your computer.
  2. 2.Open the Zenmap application.
  3. 3.In the "Target" field, enter the IP address or hostname of the target network or host that you want to scan.
  4. 4.Select the type of scan you want to perform from the "Profile" drop-down menu. For example, you could select the "Intense Scan" profile to perform a comprehensive scan of the target.
  6. 5.Optionally, you can set additional scanning options, such as the ports to scan, the level of verbosity, and the speed of the scan.
  7. 6.Click the "Scan" button to start the scan.
  8. 7.Once the scan is complete, Zenmap will display the results in the interface, where you can view them in various formats and perform further analysis.


Short IP Scanner

A Short IP Scanner is a tool used to scan a network for active hosts and IP addresses. It sends out ICMP echo requests (pings) to a range of IP addresses and waits for a response to determine which IP addresses are active and which are not.

Usage example:

fping -g 192.168.1.0/24


Advanced Port Scanner

Advanced Port Scanner is a free network scanner tool that is used to identify open ports on network devices and detect running services on those ports. It is a Windows-based tool that is easy to use and offers advanced features such as remote shutdown and Wake-on-LAN.

Usage example:

c:\> AdvancedPortScanner.exe /host:192.168.0.1 /range:1-100 /export:c:\results.csv /profile:"My Scan Profile"


Wireshark

Wireshark is a free and open-source network protocol analyzer that can capture and display network traffic in real-time. It is a powerful tool for analyzing network traffic and identifying potential security vulnerabilities.

Here's an example of how to use Wireshark:

  1. 1.Download and install Wireshark on your computer.
  2. 2.Open the program and select the network interface you want to capture traffic from.
  3. 3.Click the "Start" button to begin capturing network traffic.
  4. 4.Once the capture is complete, you can analyze the traffic in real-time or save it for later analysis.
  5. 5.Use the built-in filters and analysis tools to identify potential security vulnerabilities or performance issues.

Here's an example of a Wireshark filter expression:

ip.addr == 192.168.0.1 && tcp.port == 80

This filter will capture only network traffic that is sent to or from the IP address 192.168.0.1 on port 80, which is the default port for HTTP traffic.


Angry IP Scanner

Angry IP Scanner is a fast and lightweight network scanner that can scan IP addresses and ports to identify active hosts and running services. It is a popular tool for network administrators and ethical hackers to perform network reconnaissance and identify potential security vulnerabilities.

Here's an example of how to use Angry IP Scanner:

  1. 1.Download and install Angry IP Scanner on your computer.
  2. 2.Open the program and enter the IP address range you want to scan.
  3. 3.Configure any additional scanning options, such as the ports to scan or the timeout values.
  4. 4.Click the "Start" button to begin the scan.
  5. 5.Once the scan is complete, you can view the results in the program's interface or export them to a file.

Here's an example of an Angry IP Scanner command-line interface command:

$ sudo angryipscanner -i 192.168.1.0/24 -p 22,80,443 -t 1000

This command will scan the IP address range 192.168.1.0/24 (all addresses in the 192.168.1.x subnet) for hosts that have ports 22 (SSH), 80 (HTTP), or 443 (HTTPS) open. The `-t` option specifies a timeout value of 1000 milliseconds.


OpenVas

OpenVAS (Open Vulnerability Assessment System) is a free and open-source vulnerability scanner that can be used to identify and prioritize security vulnerabilities in a network. It is often used by ethical hackers and security professionals to assess the security posture of a network and identify potential vulnerabilities before they can be exploited by attackers.

Here's an example of how to use OpenVAS:

  1. 1.Download and install OpenVas on your computer.
  2. 2.Open the program and configure the target network or host that you want to scan.
  3. 3.Configure any additional scanning options, such as the types of vulnerabilities to scan for or the severity levels to prioritize.
  4. 4.Click the "Start Scan" button to begin the scan.
  5. 5.Once the scan is complete, you can view the results in the program's interface or export them to a file.

Here's an example of a command-line interface command to start an OpenVAS scan:

$ omp -u admin -w password -h 127.0.0.1 -p 9390 -T 10

This command will start an OpenVAS scan on the target host 127.0.0.1 on port 9390 using the default admin credentials (-u admin -w password). The `-T` option specifies the scan type, with 10 representing a full and comprehensive scan.


Nessus

Nessus is a commercial vulnerability scanner developed by Tenable Network Security that can be used to identify and prioritize security vulnerabilities in a network. It is widely used by organizations and security professionals to assess the security posture of a network and identify potential vulnerabilities before they can be exploited by attackers.

Here's an example of how to use Nessus:

  1. 1.Download and install Nessus on your computer.
  2. 2.Open the program and configure the target network or host that you want to scan.
  3. 3.Configure any additional scanning options, such as the types of vulnerabilities to scan for or the severity levels to prioritize.
  4. 4.Click the "Start Scan" button to begin the scan.
  5. 5.Once the scan is complete, you can view the results in the program's interface or export them to a file.

Here's an example of a command-line interface command to start a Nessus scan:

$ nessuscli scan new "My Scan" -T "Target Networks" 192.168.1.0/24 -p 1-65535

This command will start a Nessus scan named "My Scan" that will scan the target network 192.168.1.0/24 for all ports (-p 1-65535).
