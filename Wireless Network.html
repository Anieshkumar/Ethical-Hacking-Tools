Reaver

Reaver is a free and open-source tool for testing the security of wireless networks by attempting to recover the WiFi password through a brute-force attack on the WPS (Wi-Fi Protected Setup) PIN. It works by sending a series of PINs to the access point or router until the correct one is found. Reaver is primarily designed to work against WPS-enabled wireless access points and is used to test the strength of WPS PINs.

Reaver is a command-line tool and can be used on Linux and Unix operating systems. Here's an example command to run Reaver:

reaver -i wlan0mon -b 00:01:02:03:04:05 -vv

In this example, "wlan0mon" is the name of the wireless interface, "00:01:02:03:04:05" is the MAC address of the target access point, and "-vv" enables verbose output. The tool then starts brute-forcing the WPS PIN to gain access to the network. It's important to note that Reaver should only be used on networks that you have permission to test.



Kismet

Kismet is a free and open-source wireless network detector, sniffer, and intrusion detection system. It can detect wireless networks and devices in its range and passively collect information from them, such as SSID, MAC address, signal strength, and channel.

Kismet supports various wireless network interfaces, including Wi-Fi, Bluetooth, and Zigbee. It can also be used with GPS receivers to record the location of detected networks and devices.


Kismet can be used for various purposes, such as network discovery, network monitoring, network troubleshooting, and wireless security auditing. It can also be integrated with other tools and systems, such as IDS/IPS, SIEM, and network management systems.


To use Kismet, you need to install it on a Linux or BSD system and configure it to use a compatible wireless network interface. You can then start Kismet and use its web-based or command-line interface to view and analyze the collected data. Kismet also provides various options and plugins to customize its functionality and output.



Wifite

Wifite is an open-source tool used for auditing and attacking wireless networks. It is designed to automate wireless network auditing, including cracking of WEP and WPA keys, and features automatic scanning and capturing of WPA handshakes. Wifite uses a combination of different tools, including Aircrack-ng, Reaver, and Pyrit, to perform various wireless attacks.

Wifite's usage is straightforward and user-friendly. It has a command-line interface that offers different options to perform attacks. To use Wifite, first, the user needs to ensure that the wireless network interface card is in monitor mode. Once that is done, the user can simply launch Wifite and choose the target network to attack. Wifite will then automatically perform a series of attacks to crack the WEP or WPA key. The user can also choose to attack a specific network using its BSSID.


It is important to note that Wifite should only be used on networks that the user has permission to test or audit, as unauthorized use of the tool may be illegal and considered unethical.



OpenWIPS-ng

OpenWIPS-ng is an open source Wireless Intrusion Prevention System (WIPS) framework that is designed to detect and prevent malicious activities on wireless networks. It provides real-time monitoring and can be used to detect rogue access points, unauthorized clients, and other potential security threats.

The tool can be used to monitor multiple wireless networks simultaneously and supports a range of wireless adapters. OpenWIPS-ng uses various detection techniques such as signature-based detection, anomaly detection, and statistical analysis to detect potential threats.


The tool has a web-based interface that allows users to view real-time statistics, configure settings, and manage alerts. It also provides the ability to generate reports and export data.


OpenWIPS-ng can be used by security professionals and network administrators to secure wireless networks and prevent unauthorized access. It is available for free and can be installed on various operating systems including Linux and Windows.



EAPHammer

EAPHammer is an open-source toolkit for performing targeted evil twin attacks against WPA2-Enterprise networks. The tool allows an attacker to bypass the mutual authentication process between a client and a server in a WPA2-Enterprise network.

The tool works by creating a fake access point that emulates the target network and sends deauthentication packets to the target client, forcing it to disconnect from the legitimate access point and connect to the fake one. EAPHammer then performs a fake EAP authentication process with the target client and captures its credentials.


The usage of EAPHammer involves configuring the tool with the necessary information about the target network, including the EAP method and other network parameters. The tool provides a simple command-line interface for executing the attack and capturing the credentials. It also offers various advanced features such as the ability to automatically scan for available networks, perform target selection, and customize the attack parameters.



Wifiphisher

Wifiphisher is a wireless network security tool that is used to launch phishing attacks against WiFi networks. The tool was created to demonstrate how easily users could be tricked into providing sensitive information such as login credentials over unsecured networks.

Wifiphisher works by creating a fake access point and then tricking connected devices into connecting to it instead of the legitimate network. The tool then prompts the user to enter their login credentials, which are then captured by the attacker.


The usage of Wifiphisher involves identifying a target WiFi network, creating a fake access point with a similar SSID and security settings, and launching a phishing attack against connected devices. The tool offers various attack scenarios such as credential harvesting, advanced phishing attacks, and man-in-the-middle attacks. It is important to note that Wifiphisher should only be used for ethical hacking purposes with the owner's permission and not for illegal activities.



Fluxion

Fluxion is an open-source wireless network security auditing tool. It automates the process of a fake Access Point (AP) impersonation and performs a Man-in-the-Middle (MITM) attack to extract the pre-shared key (PSK) password. It is mainly used for testing and demonstrating security weaknesses in wireless networks. Fluxion is capable of performing both passive and active scanning, and can crack WPA and WPA2 encryption methods. The tool is written in Bash and Python programming languages.

To use Fluxion, the following steps can be taken:

  1. 1.Download and install Fluxion on a Linux system.
  2. 2.Run Fluxion and select the target wireless network to attack.
  3. 3.Choose the attack method and wait for the target to connect to the fake AP.
  4. 4.Extract the PSK password and access the target network.


Bettercap

Bettercap is a Swiss army knife for network attacks and monitoring. It is an open-source network utility that allows for network monitoring, injection, and manipulation. It can perform various functions such as ARP spoofing, DNS spoofing, and Man-in-the-middle attacks.

Some of the main features of Bettercap include:

  1. 1.Full sniffing of HTTP/HTTPS/TLS traffic, with SSLsplit-like features.
  2. 2.Various sniffing modes to suit different needs.
  3. 3.Full IPv6 support.
  4. 4.ARP spoofing and ARP poisoning.
  5. 5.Network discovery and scanning.
  6. 6.Built-in packet editors and custom filters.
  7. 7.Session hijacking and cookie injection.
  8. 8.MITM proxy support.
